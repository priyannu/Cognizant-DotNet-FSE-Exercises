## Data Structures and Algorithms (Java)

### Exercise 2 – E-Commerce Platform Search Function

Implemented search operations on a product catalog.

**Algorithms Used**

* Linear Search
* Binary Search

**Complexity Analysis**

Linear Search

* Best Case: O(1)
* Average Case: O(n)
* Worst Case: O(n)

Binary Search

* Best Case: O(1)
* Average Case: O(log n)
* Worst Case: O(log n)
